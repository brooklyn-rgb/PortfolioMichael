# Michael Mujabi - Portfolio

This is the portfolio website of Michael Mujabi, a Senior ASP.NET & Full-Stack Developer.

## Features

- Modern, responsive design
- Contact form with email sending
- Project showcase
- Professional experience timeline

## Deployment

The project is deployed on Render.com.

## Environment Variables

The following environment variables are required:

- `GMAIL_APP_PASSWORD`: The app password for the Gmail account used to send emails.
